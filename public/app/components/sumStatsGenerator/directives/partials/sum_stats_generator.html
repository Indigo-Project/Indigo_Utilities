Instructions
  1. Export any 'Trimetrix' or 'Talent Insights' assessment data from TTI. Currently, this function only works with 'Trimetrix' and 'Talent Insights' assessments, and will not accept other assessment/instrument types.
  2. Click 'Choose File' to load .csv files one at a time, then 'Add Report'. Multiple .csv files can be processed at one time. Add all reports for each school.
  3. Enter the output file name, and click 'Generate Summary Stats' to generate the summary statistics report .csv for the given population.
  4. The Summary Statistics Report Currently Includes School Averages, School Standard Deviation, and Adult Averages for DISC and Motivators.
  5. If you would like identify the .csv with the name of the school, do so when naming the file, or Replace the word 'School' with the school name in the Average and Standard Deviation Titles. (rows 3 & 4).
